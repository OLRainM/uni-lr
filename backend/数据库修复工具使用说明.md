## 📝 执行的操作

### 1. 重命名表（去掉tb_前缀）
- `tb_user` → `users`
- `tb_category` → `categories`
- `tb_product` → `products`
- `tb_cart` → `cart`
- `tb_order` → `orders`
- `tb_order_item` → `order_items`
- `tb_address` → `addresses`
- `tb_knowledge` → `knowledge`

### 2. 删除deleted字段
从以下表中删除逻辑删除字段：
- `users`, `products`, `orders`, `addresses`, `knowledge`

### 3. 创建缺失的表
- `coupons` - 优惠券表
- `user_coupons` - 用户优惠券表
- `banners` - 轮播图表

### 4. 插入测试数据
- 3条优惠券数据
- 3条轮播图数据

## ⚠️ 注意事项

### 1. 执行前确认
- ✅ 确保数据库可以连接
- ✅ 确保有足够的权限（ALTER TABLE, DROP, CREATE）
- ✅ 建议先备份数据库（虽然操作是安全的）

### 2. Python环境
- 需要 Python 3.6+
- 会自动安装 `pymysql` 库

### 3. 执行时间
- 通常在 10-30 秒内完成
- 如果表数据量大，可能需要更长时间

### 4. 错误处理
脚本会智能处理以下情况：
- 表已重命名 → 跳过
- 字段已删除 → 跳过  
- 表已存在 → 跳过

## 🐛 常见问题

### 问题1：提示"未检测到Python"

**解决方法**：
1. 下载安装Python：https://www.python.org/downloads/
2. 安装时勾选 "Add Python to PATH"
3. 重新打开命令行窗口

---

### 问题2：pip install 失败

**解决方法1**：使用国内镜像
```bash
pip install pymysql -i https://pypi.tuna.tsinghua.edu.cn/simple
```

**解决方法2**：升级pip
```bash
python -m pip install --upgrade pip
pip install pymysql
```

---

### 问题3：数据库连接失败

**检查清单**：
1. ✅ 数据库服务器是否运行
2. ✅ IP地址是否正确：115.190.125.177
3. ✅ 端口是否正确：3306
4. ✅ 用户名密码是否正确
5. ✅ 防火墙是否允许连接

**测试连接**：
```bash
mysql -h 115.190.125.177 -P 3306 -u tibetan_app -p
# 输入密码：LR-project-7788
```

---

### 问题4：权限不足

**错误信息**：`Access denied` 或 `Permission denied`

**解决方法**：
确保 `tibetan_app` 用户有以下权限：
```sql
GRANT ALTER, CREATE, DROP, SELECT, INSERT, UPDATE 
ON tibetan_medicine.* 
TO 'tibetan_app'@'%';
FLUSH PRIVILEGES;
```

---

### 问题5：某些表已存在

**这是正常的！** 脚本会自动跳过已存在的表。

显示信息：
```

```
