# 🚀 快速开始 - 3步修复数据库

---

## 第1步：双击运行批处理文件

找到并双击文件：
```
backend/运行数据库修复.bat
```

---

## 第2步：确认执行

看到提示时输入 `y` 并按回车：
```
是否继续？(y/n): y
```

---

## 第3步：重启后端服务

**在IDEA中**：
1. 点击红色停止按钮 🛑
2. 点击绿色运行按钮 ▶️

**或在命令行**：
```bash
# 停止：Ctrl+C
# 启动：
cd backend
mvn spring-boot:run
```

---

## 💡 如果Python未安装

1. 访问：https://www.python.org/downloads/
2. 下载并安装Python 3.x
3. 安装时勾选 **"Add Python to PATH"**
4. 重新打开命令行，再次运行批处理文件
