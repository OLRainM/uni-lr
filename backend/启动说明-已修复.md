# 🎉 Spring Boot 项目启动说明（已修复兼容性问题）

> 使用最新稳定版本配置，已解决 Spring Boot 3.x 与 MyBatis-Plus 的兼容性问题

---

## ✅ 已完成的修复

### 关键改动

1. **升级 Spring Boot**：`3.1.5` → **`3.3.7`**（最新稳定版）

2. **使用 Spring Boot 3 专用的 MyBatis-Plus**：
   - ❌ 旧版：`mybatis-plus-boot-starter` 3.5.5
   - ✅ 新版：**`mybatis-plus-spring-boot3-starter` 3.5.9**
   - 这是关键！Spring Boot 3.x 必须使用 `spring-boot3-starter` 版本

3. **移除不兼容的依赖**：
   - ❌ 移除：`spring-boot-starter-data-jpa`
   - ❌ 移除：`spring-boot-starter-security`
   - 这两个依赖在当前项目中不需要，反而会导致冲突

4. **切换数据库连接池**：
   - ❌ 旧版：HikariCP（默认）
   - ✅ 新版：**Druid**（更强大的监控和配置）

5. **升级 Lombok**：`1.18.30` → **`1.18.42`**

6. **切换 JWT 库**：
   - ❌ 旧版：`jjwt` (io.jsonwebtoken)
   - ✅ 新版：**`java-jwt` (com.auth0)** - 更简单易用

## 🚀 启动步骤

### Step 1: 刷新 Maven 依赖（必须！）

1. **打开 IDEA 右侧的 Maven 面板**
   - 点击 **Maven** 标签
   - 或按 `Ctrl + E` → 输入 "Maven"

2. **刷新项目**
   - 点击刷新按钮 🔄 **Reload All Maven Projects**
   - **等待下载新的依赖**（约2-3分钟）
   - 右下角会显示进度

3. **确认下载完成**
   - 看到 "Indexed XXX files"
   - 没有错误提示

---

### Step 2: 清理旧的构建文件

在 IDEA 菜单中：

```
Build → Clean Project
```

等待清理完成。

---

### Step 3: 重新构建项目

```
Build → Rebuild Project
```

等待构建完成（约30秒-1分钟）。

---

### Step 4: 运行项目

1. **找到启动类**
   ```
   TibetanMedicineApplication.java
   ```

2. **运行**
   - 按 **Shift + F10**
   - 或右键 → Run 'TibetanMedicineApplication'

3. **查看启动日志**

## 🐛 如果还有问题

### 问题1：依赖下载失败

**解决**：

1. 检查网络连接
2. 尝试使用国内 Maven 镜像（阿里云）
3. 在 IDEA 中：File → Settings → Maven → 配置 mirrors

### 问题2：端口被占用

**错误**：`Port 8080 was already in use`

**解决**：

```powershell
# 查找进程
netstat -ano | findstr :8080

# 结束进程
taskkill /PID 进程号 /F
```

### 问题3：Redis 连接失败

**错误**：`Unable to connect to Redis`

**解决**：

- Redis 是可选的，如果没有安装可以暂时注释掉 Redis 配置
- 或安装 Redis：`winget install Redis`

```xml
<dependency>
    <groupId>com.alibaba</groupId>
    <artifactId>druid-spring-boot-starter</artifactId>
    <version>1.2.18</version>
</dependency>
```
