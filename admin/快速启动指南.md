## 💡 使用技巧

### 商品管理技巧

**添加商品**：
1. 点击右上角"添加商品"按钮
2. 填写商品信息（名称、分类、价格、库存）
3. 上传商品图片
4. 填写商品简介和详情
5. 选择是否上架
6. 点击"确定"保存

**批量操作**：
- 使用搜索筛选找到目标商品
- 快速切换上下架状态

### 订单管理技巧

**快速发货**：
1. 在订单列表找到"待发货"订单
2. 点击"发货"按钮
3. 选择物流公司
4. 输入物流单号
5. 确认发货

**订单筛选**：
- 按订单号精确查找
- 按客户姓名模糊搜索
- 按订单状态筛选

### 数据分析技巧

**查看趋势**：
- 订单趋势图：了解每日订单量变化
- 收入统计图：掌握每日收入情况
- 最近订单：关注实时订单动态

## 🔧 常见问题

### Q1：启动失败怎么办？

**解决方案**：
1. 检查 Node.js 版本（需要 >= 16.0.0）
```bash
node -v
```

2. 清除缓存重新安装
```bash
rm -rf node_modules package-lock.json
npm install
```

3. 检查端口是否被占用

---

### Q2：登录失败怎么办？

**检查项**：
- 用户名：admin
- 密码：123456（必须完全一致）
- 密码至少6位

---

### Q3：页面空白怎么办？

**解决方案**：
1. 打开浏览器控制台查看错误
2. 确认依赖已正确安装
3. 刷新页面或清除浏览器缓存

---

### Q4：如何退出登录？

**操作步骤**：
1. 点击右上角用户头像
2. 选择"退出登录"
3. 确认退出
