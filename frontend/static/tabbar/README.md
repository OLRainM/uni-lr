# TabBar 图标说明

## 需要的图标文件

请在此目录下添加以下 8 个图标文件：

### 1. 首页图标
- `home.png` - 首页默认图标（未选中）
- `home-active.png` - 首页选中图标

### 2. 分类图标
- `category.png` - 分类默认图标（未选中）
- `category-active.png` - 分类选中图标

### 3. 购物车图标
- `cart.png` - 购物车默认图标（未选中）
- `cart-active.png` - 购物车选中图标

### 4. 我的图标
- `user.png` - 我的默认图标（未选中）
- `user-active.png` - 我的选中图标

## 图标规格要求

- 尺寸：81px * 81px（推荐）
- 格式：PNG（支持透明背景）
- 大小：每个图标不超过 40KB
- 颜色：
  - 未选中状态：灰色 (#999999)
  - 选中状态：棕色 (#8B4513) - 藏药主题色

## 图标资源获取

可以从以下渠道获取图标：
- [iconfont.cn](https://www.iconfont.cn/) - 阿里巴巴矢量图标库
- [iconfinder.com](https://www.iconfinder.com/) - 免费图标资源
- [flaticon.com](https://www.flaticon.com/) - 扁平化图标
- Figma/Sketch 等设计工具自行绘制

## 临时解决方案

在图标准备好之前，可以临时使用纯色方块作为占位符，或者使用 uni-ui 提供的图标组件。
