## 🚀 运行说明

### 1. 环境准备

```bash
# 安装依赖
- JDK 17+
- MySQL 8.0+
- Maven 3.6+
```

### 2. 数据库初始化

```bash
# 执行SQL脚本
mysql -u root -p < backend/src/main/resources/sql/tibetan_medicine.sql
```

### 3. 配置修改

修改 `application.yml` 中的数据库连接信息：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/tibetan_medicine
    username: root
    password: your_password
```

### 4. 启动服务

```bash
cd backend
mvn spring-boot:run
```

服务启动后访问：http://localhost:8080/api

### 5. 前端对接

前端配置已更新，`utils/request.js` 中的 `BASE_URL` 已指向后端服务：

```javascript
const BASE_URL = 'http://localhost:8080/api'
```
