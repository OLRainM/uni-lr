## 二、关于已废弃的 admin/ 目录

`admin/` 目录下原本是一个基于 Vue 3 + Element Plus + Vite 的管理后台项目，
登录方式为账号密码（admin/123456），使用模拟数据（mock），**现已被废弃**。

**替代方案**：`adminWeb/` 目录，纯 HTML+CSS+JS 实现，
随 Spring Boot 后端启动，秘钥登录，调用真实后端接口。
访问地址：`http://localhost:8080/api/admin-web/index.html`
