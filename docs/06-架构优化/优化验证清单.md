### 测试需要登录的接口

```javascript
// 先登录获取token
uni.request({
  url: 'http://localhost:8080/api/user-v2/login',
  method: 'POST',
  data: {
    code: 'test_code_123'
  },
  success: (res) => {
    const token = res.data.data.token
    uni.setStorageSync('token', token)
    
    // 然后测试需要登录的接口
    uni.request({
      url: 'http://localhost:8080/api/user-v2/info',
      method: 'GET',
      header: {
        'Authorization': 'Bearer ' + token
      },
      success: (res) => {
        console.log('用户信息:', res.data)
      }
    })
  }
})
```

**预期结果**：
- [ ] 登录成功，获取到token
- [ ] 获取用户信息成功

## ❌ 如果遇到问题

### 问题1：启动报错

**检查**：
- [ ] JDK版本是否为17
- [ ] Maven依赖是否刷新成功
- [ ] 数据库是否启动
- [ ] 端口8080是否被占用

### 问题2：访问/doc.html显示404

**检查**：
- [ ] 项目是否启动成功
- [ ] 地址是否正确：http://localhost:8080/api/doc.html
- [ ] Knife4j依赖是否添加成功

### 问题3：接口返回401

**检查**：
- [ ] 是否需要登录的接口
- [ ] 是否添加了@SkipAuth注解
- [ ] token是否正确
- [ ] token格式是否为：Bearer xxx

### 问题4：参数校验不生效

**检查**：
- [ ] Controller方法参数是否添加了@Valid
- [ ] DTO字段是否添加了校验注解
- [ ] spring-boot-starter-validation依赖是否存在
