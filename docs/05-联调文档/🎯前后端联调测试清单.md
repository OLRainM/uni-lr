## 🔧 需要修复的问题

### 高优先级 🔴

#### 1. 优惠券页面集成API
**文件**：`src/pages/coupon/list.vue`

**当前问题**：使用硬编码测试数据

**修复方案**：
```javascript
import { getMyCoupons } from '@/api/index.js'
import { onMounted } from 'vue'

const loadCoupons = async () => {
    try {
        const res = await getMyCoupons()
        if (res && res.length > 0) {
            // 根据status分类
            availableCoupons.value = res.filter(item => item.status === 0)
            usedCoupons.value = res.filter(item => item.status === 1)
            expiredCoupons.value = res.filter(item => item.status === 2)
        }
    } catch (e) {
        console.error('加载优惠券失败:', e)
    }
}

onMounted(() => {
    loadCoupons()
})
```
