## 🎯 前端使用注意事项

### 知识库列表
```javascript
// ❌ 错误
image: item.image

// ✅ 正确
image: item.cover
```

### 订单状态显示
```javascript
// ✅ 支持数字和字符串两种格式
const statusMap = {
    0: '待付款',
    1: '待发货',
    2: '待收货',
    3: '已完成',
    4: '已取消',
    // 同时支持字符串格式
    'unpaid': '待付款',
    'unshipped': '待发货',
    // ...
}
```

### 商品字段
```javascript
// ✅ 正确的字段名
{
    subtitle: item.subtitle,  // 副标题
    image: item.image,        // 主图
    params: item.params,      // 参数
    specs: item.specs         // 规格
}
```
