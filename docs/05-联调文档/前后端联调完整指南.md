## 🎯 API接口数据格式规范

### 统一返回格式
```javascript
{
  code: 0,           // 0-成功，其他-失败
  message: "success",
  data: {
    // 单条数据直接返回对象
    // 列表数据返回 { list: [], total: 0, pageNum: 1, pageSize: 10 }
  }
}
```

### 分页数据格式
```javascript
{
  code: 0,
  message: "success",
  data: {
    list: [...],      // 数据列表
    total: 100,       // 总数
    pageNum: 1,       // 当前页
    pageSize: 10      // 每页数量
  }
}
```
